Havens

The idea of Havens is to let you build your own social media. It's a club-based community allowing you connect with people who share your passions.

Havens is design using React (Next JS will be implemented soon to enhance the rendering) for the front and it is using the Havens API to deal with the database.

This website is aim to grow, and I'm using all my free time trying to implement new functionnalities. If you interested in the project don't hesitate to reach me here: admin@havens.fr

Havens API

Havens REST API is build using Express in NodeJS, the documentation of this API is not yet available. The goal of this API is to let any applications interact with the Havens social network for free.

mydreamwallet

mydreamwallet was one of the first website I made, the purpose of mydreamwallet was to simulate crypto investemnts, you could invest fictive money into cryptos and see how your investemnts would grow or not in the future.

Later on, I also implemented a card trading collection, which kinda looks like a NFT's market place where you could trade with your friends. Each day you could roll 3 of them for free.

This website was built using Html, Css and Php for the backend, with just a little of Javascript to use the websockets of binance in order to get the crypto price at the moment.
